# How to run the project
# Prerequisite
gradle and kotlin should be installed.

# Steps for running
1. Clone the project
2. Load porject in IntelliJ
3. Run following command on terminal of IntelliJ to build
./gradlew clean build
4. Run following command on terminal of IntelliJ to start app
./gradlew bootrun

# How to terminate instance
Ctrl C on terminal

# Postman
Http Method: POST
URL: http://localhost:8080/api/validate
Header: Content-Type: application/json
Request: Check src/sample/request.json
